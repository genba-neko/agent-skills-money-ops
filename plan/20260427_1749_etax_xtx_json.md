# e-Tax xtx → JSON 化基盤 + 仕様書調査

## 対象 issue

[#59](https://github.com/genba-neko/agent-skills-money-ops/issues/59)

---

## 目的

1. **各年の確定申告情報を記録として残す** — 過去申告の永続保管・参照
2. **確定申告の自動入力** — 来年度の `tax-etax` (e-Tax 作成コーナー入力) での値転記、`tax-calc` / `tax-review` での前年値参照・差分比較

---

## 背景

### 復元可能性調査結果 (実測)

| ファイル | 形式 | 復元可能性 |
|---|---|---|
| `r6syotoku_kessan_*.data` | NTA 作成コーナー独自形式 (マジック `eMHO`) | × 暗号化、仕様非公開 |
| `r6syotoku_*.pdf` | PDF | △ テキスト抽出可能だが座標解析必要 |
| `JyusinData*.xtx` | **平文 UTF-8 XML、84KB** | **◎ そのまま JSON 化可能** |

→ **xtx を一次データソースとして採用**

### サンプル xtx 構造 (R6 所得税)

- namespace: `http://xml.e-tax.nta.go.jp/XSD/shotoku` (所得税)
- フォーム版: VR=24.0.0
- 要素数: 2527
- 帳票 (form id):
  - `KOA020-1` 第一表
  - `KOA050-1` 第二表
  - `KOA210-1` 財産債務調書
  - `KOB060-1`, `KOB060-2` 青色申告決算書
  - `KOB240-1`, `KOB550-1`, `KOB560-1`, `KOB565-1`, `KOB975-1` 青色関連明細
  - `KOC080-1` 分離課税 第三表 (株式譲渡)
  - `KOE040-1` 明細 (寄附金 or 医療費)
  - `TEA060-1` 添付書類
  - `TEG700.../TEG830...` 添付書類 (受信通知等)
  - `IT` ヘッダ (税務署/受付情報)
- フィールドコード体系: BFC / DHC / AFL / WML / BXI / EHQ / AMF / 他多数
- XMLDSig 署名付き (`dsig:Transform`)

ユーザーの確定申告には以下が含まれる:
- 青色申告 (損益・貸借・月別売上)
- 医療費控除 (明細書)
- 株式関連 (分離課税・譲渡所得)
- 寄附金控除 (ふるさと納税)

## ファイル配置

### データ (コミット禁止: `data/**`)

```
data/etax/<year>/
├── raw/                           # ユーザーが手動配置
│   ├── JyusinData*.xtx            # e-Tax 提出済データ (一次ソース)
│   ├── r*syotoku_*.pdf            # 提出前/提出版 PDF (参考保管)
│   └── r*syotoku_kessan_*.data    # 作成コーナー保存形式 (参考保管・暗号化)
└── normalized.json                # xtx → 素 JSON
```

### コード

```
src/money_ops/etax/
├── __init__.py
├── parser.py                       # xtx → NormalizedReturn dataclass + JSON
└── mapping/
    └── <year>/                     # 帳票項目仕様書ベース mapping (本 issue 雛形のみ)
        └── README.md               # 取得元・整備方針

skills/tax-etax/
└── (本 issue では parser CLI のみ)
```

CLI 候補:
```bash
python skills/tax-etax/parse_xtx.py --year 2024
```

```json
  "schema_version": "1.0",
  "tax_type": "shotoku",
  "form_version": "24.0.0",
  "year": 2024,
  "source_file": "JyusinData20250314082143897313.xtx",
  "header": { "...": "IT 帳票 (税務署/受付情報)" },
  "forms": {
    "KOA020-1": { "BFC00000": "...", "BFC00010": "..." },
    "KOA050-1": { "AFL00000": "..." },
    "KOB060-1": { ... },
    "KOC080-1": { ... }

  "attachments": [ "TEG700...", "TEG830..." ]
```

```
├── KOA050.json
└── ...
```

## 注意事項

- xtx は **氏名・住所・マイナンバー・所得金額** を含む可能性大 → ログ出力時の慎重さ要 (期間中の標準出力でも個人情報を出さない)
- `data/` 既存ルール (コミット禁止) で xtx 自動カバー
- `.work/` 配下のパスはプラン・コード・コメントに記載禁止
- 1 サンプル (R6) のみで設計するが、年度 (R5/R7) 違いがあれば構造差を後追いで吸収
- XMLDSig 署名検証は本 issue では実施しない (改ざん検出用、自家保管では不要)
